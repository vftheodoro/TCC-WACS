Sobre o Projeto

O WACS (Wheelchair Automation Control System) é uma solução inovadora de mobilidade para pessoas com deficiência (PCDs) desenvolvida na Etec de Registro como projeto de TCC do curso de Desenvolvimento de Sistemas.

Nossa missão é promover a inclusão e autonomia através da tecnologia, oferecendo uma cadeira de rodas automatizada controlada via smartphone e um aplicativo colaborativo de mapeamento urbano acessível.
