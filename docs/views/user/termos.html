Termos de Uso do Projeto WACS (TCC)

Bem-vindo ao WACS (Wheelchair Automation Control System), um projeto acadêmico de Trabalho de Conclusão de Curso (TCC) da Etec de Registro. Este documento descreve as condições de uso do sistema em fase de testes. Ao criar uma conta ou utilizar o WACS, você concorda com estes Termos de Uso.

1. Sobre o Projeto

O WACS é um protótipo desenvolvido para fins de pesquisa e aprendizagem, sem caráter comercial. Todas as funcionalidades estão em fase de testes e podem ser alteradas conforme o andamento do TCC.

2. Cadastro e Segurança dos Dados

  • Para usar o sistema, é necessário criar uma conta com username, e-mail e senha.
  • As informações pessoais fornecidas (nome, cidade, tipo de mobilidade, comorbidades e foto de perfil) são opcionais, mas ajudam na experiência colaborativa.
  • E‑mails e senhas são armazenados de forma segura e secreta na plataforma Firebase, com criptografia e acesso restrito apenas à equipe do projeto.

3. Uso do Sistema

  • Você pode avaliar locais, adicionar informações e interagir com outros usuários.
  • Este é um ambiente de testes: bugs e instabilidades podem ocorrer.
  • Conteúdos ofensivos ou discriminatórios não são permitidos.

4. Privacidade

  • Sua privacidade é importante: não compartilhamos seus dados com terceiros.
  • Você pode editar ou excluir suas informações a qualquer momento no perfil.

5. Responsabilidades

  • Este projeto é fornecido "como está" para fins de pesquisa. Não oferecemos garantias de disponibilidade contínua.
  • A equipe TCC não se responsabiliza por perdas de dados decorrentes de mau uso ou instabilidades.

6. Encerramento de Conta

  • Você pode encerrar sua conta a qualquer momento pela seção de configurações.
  • Em caso de desativação, seus dados serão removidos do sistema.

7. Contato

Para dúvidas ou sugestões sobre o TCC, envie e-mail para: projeto.wacs@etecregistro.edu.br
