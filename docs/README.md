# WACS - Wheelchair Automation Control System - Website

This directory (`docs/`) contains the source files for the official website of the WACS project.

## Project Description

WACS (Wheelchair Automation Control System) is an innovative mobility solution aimed at enhancing the autonomy and accessibility for individuals with physical disabilities. The project encompasses:

*   **Automated Wheelchair Control:** A system allowing users to control their wheelchairs via a smartphone application.
*   **Collaborative Accessibility Mapping:** A feature within the application that enables users to find and share information about accessible locations.
*   **Community Features:** Tools to help users connect and share experiences.

This website serves as the main landing page and information hub for the WACS project, showcasing its features, team, and progress.
